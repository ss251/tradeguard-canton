# research/ — process & decision archive

This directory is the **paper trail** behind TradeGuard, not part of the shipped product.
It captures how we got from a blank brief to the final build: the Grok research passes,
the idea-selection and pivot decisions, and the early planning.

**Naming note:** earlier docs here refer to the project as **PrivyFactor** (a private
invoice-factoring concept that was the initial lead). We deliberately pivoted to
**TradeGuard** — private multilateral netting + atomic settlement — see
[`q5_pivot_validation.md`](q5_pivot_validation.md). Some docs also scope tech we
researched but didn't ship (cn-quickstart, PQS); the shipped stack is Canton Builder
LocalNet + JSON Ledger API v2.

For the actual product, start at the top-level [`README.md`](../README.md).

## Contents
- `STRATEGY.md`, `SYNTHESIS.md` — research synthesis + the idea bet (pre-pivot)
- `PIPELINE.md`, `OPERATIONS.md` — planning / how-we-worked
- `q1..q5_*.md` — the raw research passes (tech stack, build ideas, winning strategy,
  landscape, pivot validation)
- `CANTON_BUILD_BIBLE.md`, `DEFI_TO_TRADFI_JOURNEY.md`, `FINANCE_GLOSSARY.md` — reference notes
